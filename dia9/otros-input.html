Formulario con Diferentes Tipos de Input

Número: Puedes introducir un número del 1 al 100 en incrementos.

Color: Muestra un selector de color para elegir uno.

Date: Abre un calendario para seleccionar una fecha.

File: Permite subir un archivo desde tu dispositivo.

Range: Un control deslizante para elegir un valor entre 0 y 100.

Hidden: Guarda el valor "datoSecreto" que se enviará con el formulario, pero no es visible.
