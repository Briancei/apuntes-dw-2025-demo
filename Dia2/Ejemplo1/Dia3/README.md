 ## Selectores CSS

 Para seleccionar un grupo de elementos, podemos usar el atributo de clase, y el selector "."
 Para seleccionar un único item, podemos usar el atributo de id, y el selector "#"

 ```css
div {
 color: blue;
 }

.alumnos {
    color: red;

}
#juancito{
    color: green;
}
```

```html
<div class="alumnos" id="juancito">Hola</div>
```
